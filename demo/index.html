Popup Component 移动端演示与 API 文档

API 一览

方法 说明 主要参数
alert(options) 弹出带按钮的提示框 title：标题
content：内容
btns：按钮数组
callbacks：按钮回调数组
tips(options, callback, color) 弹出带确认按钮的提示 title、content、callback、color
msg(content, options) 消息提示（自动消失） content：消息内容
time：显示时长
icon：图标类型
confirm(info, obj, cb1, cb2) 确认框 info：内容和标题
obj：按钮和颜色配置
cb1、cb2：回调
showImagePreview(options) 图片预览弹窗 imgUrl：图片地址
onClose：关闭回调
showBottomPopup(options) 底部弹窗 title、content、btns、callbacks
load(options) 加载动画弹窗 width、height、color
close() 关闭弹窗 无

使用示例

alert 弹窗： 

popup.alert({
  title: '提示',
  content: '这是一个 alert 弹窗',
  btns: ['确定', '取消'],
  callbacks: [
    () => alert('点击了确定'),
    () => alert('点击了取消')
  ]
});
tips 弹窗： 

popup.tips({
  title: '温馨提示',
  content: '这是一个tips弹窗！'
}, () => alert('点击了确定'), ['#4caf50']);
msg 消息： 

popup.msg('这是一个消息提示', { time: 1500 });
confirm 确认框： 

popup.confirm(
  { title: '确认操作', content: '你确定要继续吗？' },
  { btn: ['确定', '取消'], color: ['#2196f3', '#21cbf3'] },
  () => alert('点击了确定'),
  () => alert('点击了取消')
);
图片预览： 

popup.showImagePreview({
  imgUrl: 'https://placekitten.com/300/200',
  onClose: () => alert('关闭图片预览')
});
底部弹窗： 

popup.showBottomPopup({
  title: '底部弹窗',
  content: '这是底部弹窗内容',
  btns: ['确定', '取消'],
  callbacks: [
    () => alert('点击了确定'),
    () => alert('点击了取消')
  ],
  cancelCallbacks: () => alert('点击了关闭')
});
loading 加载： 

popup.load({
  width: 80,
  height: 80,
  color: '#1976d2'
});
setTimeout(() => popup.close(), 2000);
